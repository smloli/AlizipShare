# AlizipShare 阿里云盘压缩文件分享工具

> ### 修改&恢复（单文件&批量）
>
> > * 拖动文件（夹）到窗口里面，回车
> >
> > * 支持直接将文件（夹）拖动到软件图标上修改
> > 
> > * 支持以右键菜单的方式修改
> >
> > * 文件每次修改，MD5都会发生变化（即使是同一个文件）
>
> ### 编译

```
go build -ldflags="-s -w" -o zipShare.exe main.go
```
